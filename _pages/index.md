A minimal Jekyll starter with no theme, configured with [Tailwind CSS](https://tailwindcss.com){:target="_blank" rel="noopener noreferrer"}, [esbuild](https://esbuild.github.io){:target="_blank" rel="noopener noreferrer"}, and [Stimulus](https://stimulus.hotwired.dev){:target="_blank" rel="noopener noreferrer"}. CSS and JavaScript are compiled automatically by a Jekyll plugin — no separate build step needed.
